 ✅ To-Do List App  

A **Flask-based To-Do List Web Application** with **User Authentication** (Register/Login), **Task Management**, **Priority Levels**, and **Due Dates**. It provides an **interactive dashboard** for managing your daily tasks efficiently.  



 ✨ Features  
✔️ **User Authentication** – Register, Login, and Logout  
✔️ **Add, Complete, Delete Tasks**  
✔️ Set **Priority (Low, Medium, High)**  
✔️ Add **Due Dates** for tasks  
✔️ Interactive **Bootstrap-based UI**  
✔️ Uses **SQLite database** for persistent storage  



 🛠 Tech Stack  
- **Backend:** Python (Flask)  
- **Database:** SQLite (via SQLAlchemy ORM)  
- **Frontend:** HTML, CSS (Bootstrap), Jinja2 Templates  
- **Security:** Password Hashing (Werkzeug)
